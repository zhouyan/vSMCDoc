Detailed Description

template<typename ResultType, std::size_t Blocks = 1>
class vsmc::AES128Engine< ResultType, Blocks >

AES-128 RNG engine.

This is a reimplementation of the algorithm AESNI engine as described in Parallel Random Numbers: As Easy as 1, 2, 3 and implemented in Random123.
