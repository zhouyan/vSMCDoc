Detailed Description

template<typename T, std::size_t K>
class vsmc::Counter< std::array< T, K > >

Using std::array of unsigned integers as counters.

This class provides methods for using std::array of unsigned integers as large integer counters.

It deals with two types of counters. The first type is in the form std::array<T, K> where T is an unsigned integer type, treated as a sizeof(T) * K * 8 bits integer. For example, std::array<std::uint32_t, 4> is treated as an 128-bits integer. The counter start with all elements being zero. The value of the integer can be calculated as, \(c_0 + c_1 M + c_2 M^2 +\cdots + c_{K-1} M^{K - 1}\), where \(c_i\) is the \(i\)th element in the std::array and has a range from zero to \(M - 1\), \(M\) is the largest number of type T plus one, that is \(2^n\) with \(n\) being the number of bits in type T.
