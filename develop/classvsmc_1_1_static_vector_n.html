Detailed Description

template<typename T, std::size_t N, typename Alloc = Allocator<T>>
class vsmc::StaticVectorN< T, N, Alloc >

Vector with statically allocated space.

This is sort of a combination of std::array and std::vector. If the template parameter is bigger than zero, then a static array is allocated. The size of the vector can be specified at construction but cannot be changed later. If the construction size n is bigger than N, dynamic memory will be used. The vector size is the maixmum of N and n.
