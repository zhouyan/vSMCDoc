4 # Changes in v2.2.0
5 
6 ## New features
7 
8 * `SingleParticle` now as iterator-like behaviors.
9 * `Particle` now has a new `sp` member that return `SingleParticle` objects.
10 * `NormalMVDistribution` and related functions for generating multivariate
11  Normal distribution random variate.
12 * New module "Random Walk" that implements generic random walk MCMC kernels and
13  Normal distribution based proposals. Both scalar and multivariate cases are
14  supported.
15 * `Covariance` is a new class that can compute the covariance matrix from
16  (weighted) samples, and output the matrix in variance formats.
17 * `cov_chol` transform a covariance matrix, stored in various formats, into the
18  lower triangular part of its Cholesky decomposition, stored as a packed row
19  major matrix
20 
21 ## Changed behaviors
22 
23 * `AlignedAllocator` template parameter `Memory` now requires its member
24  functions to be static and the class stateless. Unless you write your
25  allocator using this template, there is no need to change client code.
26 * `MatrixOrder` is now an alias to `MatrixLayout`.
27 * CBLAS and LAPACK are now required dependency.
28 
29 ## Removed features
30 
31 * `U01LRDistribution` is removed, while `U01CCDistribution` etc., remain.
32 * `UniformRealLRDistribution` and `UniformRealCCDistribution` etc., are
33  removed.
34 * Path sampling support is removed. It can be easily done through the more
35  general `Monitor`. See the GMM example.

40 # Changes in v2.1.0
41 
42 ## Important changes
43 
44 The mutation (MCMC) steps are now performed after the resampling of
45 initialization step.

47 ## Removed features

50  added back in future with a new interface.

55 # Changes in v2.0.0
56 
57 The version 2.0.0 is a major restructuring of the library since the initial
58 release. Its documentation is still been updated. For the time being, one can
59 keep using the old JSS release or the reference manual below.
60 
61 The most important change is that C++11, both language and libraries are now
62 required.
