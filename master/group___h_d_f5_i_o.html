Save a data frame in the HDF5 format from an iterator to iterators.

A data frame is similar to that in R. It is much like a matrix except that each column will be stored seperatedly as an variable and given a name.

Template Parameters
TType of the data
InputIterIterThe input iterator type, which points to input iterators
SInputIterThe input iterator type of names
Parameters
nrowNumber of rows
ncolNumber of columns
file_nameName of the HDF5 file
data_nameName of the data frame
firstAn iterator points to a sequence of iterations of length ncol. Each derefence of the iterator is iteself an iterator that points to the beginning of a column of the data frame.
sfirstAn iterator points to the beginning of a sequence of strings that store the names of each column. The dereference need to be convertible to std::string
appendIf true the data is appended into an existing file, otherwise save in a new file

Save a data frame in the HDF5 format from tuple of iterators.

A data frame is similar to that in R. It is much like a matrix except that each column will be stored seperatedly as an variable and given a name.

Parameters
nrowNumber of rows
file_nameName of the HDF5 file
data_nameName of the data frame
firstA std::tuple type object whose element types are iterators. Each element is the beginning of a column of the data frame.
sfirstAn iterator points to the beginning of a sequence of strings that store the names of each column. The dereference need to be convertible to std::string
appendIf true the data is appended into an existing file, otherwise save in a new file
