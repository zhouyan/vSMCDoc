Detailed Description

template<typename ResultType, std::size_t Rounds = 7, std::size_t Blocks = 1>
class vsmc::ARSEngine< ResultType, Rounds, Blocks >

ARS RNG engine.

This is a reimplementation of the ARS engine as described in Parallel Random Numbers: As Easy as 1, 2, 3 and implemented in Random123.
